AgriSense — IoT para viveros

Monitoreo ambiental y riego automático con ESP32 y tablero web.

Flutter Python ESP32 MQTT Timeseries Grafana
Demo (próximamente)Esquema IoT

Problema

Riego manual ineficiente y falta de datos históricos para decidir.

Solución

Sensores de humedad/temperatura con ESP32, envío MQTT y backend Python; app móvil Flutter.

Resultados

Ahorro de agua y mejor salud de plantas; alertas push y panel en tiempo real.
