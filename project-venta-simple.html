Venta Simple — Sistema de gestión

Stock, ventas, tickets, KPIs y panel web sincronizado en la nube.

Java Next.js MySQL Auth REST API Deploy
Demo (próximamente)Repo privado

Problema

Negocio con múltiples puntos de venta sin visibilidad unificada y procesos manuales lentos.

Solución

Arquitectura full‑stack con backend en Java y panel Next.js; sincronización y backups automáticos; auth por roles.

Resultados

Reduje tiempos de carga/venta y eliminé duplicados. KPIs diarios y trazabilidad completa.
