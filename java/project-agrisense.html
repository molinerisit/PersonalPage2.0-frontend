🌱 ESP32 · MQTT · Spring Boot Riego automatizado por umbrales

Problema

Riego manual ineficiente, sin histórico de variables y sin alertas tempranas.

Solución

  • Nodos ESP32 con sensores (humedad/temperatura).
  • Transporte MQTT; backend en Java / Spring Boot para la ingesta.
  • Time-series + panel Grafana; app móvil.
  • Automatización de riego con umbrales.

Resultados

Ahorro de agua, mejor salud de plantas y decisiones con datos históricos.

Stack

  • ESP32 + sensores (DHT/Capacitivo)
  • MQTT (Mosquitto/EMQX)
  • Java + Spring Boot (ingesta)
  • App móvil
  • Time-series + Grafana
